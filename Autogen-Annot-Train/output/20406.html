In addition to FVIII activity, this IS has the following values assigned: FVIII Antigen (FVIII:Ag), von Willebrand Factor Antigen (VWF:Ag), von Willebrand Factor Ristocetin Cofactor function (VWF:RCo), VWF binding to recGPIb - ristocetin-dependent (VWF:GPIbR), VWF binding to recGPIb - gain-of-function mutant (VWF:GPIbM), von Willebrand Factor Collagen Binding function (VWF:CB), and von Willebrand Factor Propeptide (VWFpp). This IS has also been assigned a FIX antigen value.
