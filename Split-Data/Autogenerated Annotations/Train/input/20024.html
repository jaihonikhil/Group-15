This table gives a summary of the BDT temperature ( T BDT ), mean chord lengths ( λ ¯ ) and the dislocation density ( ρ D ) before and after heat treatment (1300 K, 6 h). The density of boundaries (surface per unit volume) with disorientation angles above 2° ( S V ) is also provided. Furthermore, the mean disorientation angle ( θ ¯ ) is given. For calculation of the fractions of low-angle boundaries ( x LAB ) and high-angle boundaries ( x HAB ), threshold angles of 2° and 15° were chosen, respectively. The fraction of CSL boundaries ( x CSL ) includes all CSL boundaries from Σ3 to Σ25 (Brandon's criterion, exponent was set to −0.5) which are also included in the fraction of high-angle boundaries. The standard deviation was calculated between values of the four EBSD scans for each of the samples respectively.
