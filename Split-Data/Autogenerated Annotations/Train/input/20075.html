Model parameters, their description and ranges for the simulation experiments.
Parameter Description Range

motivtype Motivation type constant, seasonal, sigmoidal, exponential, sinusoidal

c gap Sigmoidal model parameter. Defines the gap (in days) between motivation changes [200, 100, 50, 30]
c decline Model specific parameter. Frequency or magnitude of motivation change [300, 270, 200, 100, 50, 40, 30, 20, 10, 5, 3, 2, 1, 0.7, 0.5, 0.2, 0.1]
c seasonality Seasonal model parameter. Rapid increase in motivation at fixed intervals [400, 365]
