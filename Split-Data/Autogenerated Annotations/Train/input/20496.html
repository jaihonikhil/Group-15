Confusion matrix of the dual-classification system used to identify objects in the VPR images. See , and references therein, for details. PD is probability of detection (%) which is based on the ratio of accurate/true automatic classifications to total (manual) identified images belonging in a category. PD is the probability that an image belonging in a class will be classified as a that taxon. SP is specificity (%), which is based on the ratio of automatic classifications actually belonging in a class to total assignments to that class. SP is the probability that the classifier's prediction is correct for each taxon. NA is not applicable. All data are counts, except SP and PD, which are percentages.
