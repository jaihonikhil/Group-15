F3D files: These are Autodesk Fusion 360 archive files including all of the parts needed for the pump and jig respectively. STL files: These are ready to print component files to be imported into any 3D-printing slicer. FORM file: This is a preconfigured print file for Formlabs Preform software. It can be used to print the entire design, with one click, on a Form 2 or Form 3 3D-printer. PRG file: Script used to control the stepper motor through the manufacturers software.
